# 1. Introduction

- **AWS DeepRacer** is a cloud-based platform that teaches **Reinforcement Learning (RL)** by training and racing autonomous 1/18-scale cars.
- The **AWS DeepRacer vehicle** is a **Wi-Fi-enabled autonomous car** that uses a **Reinforcement Learning (RL)** model to drive itself on a physical track.

# 2. How It Works

1. Train an RL model.
2. Deploy the model to a virtual or physical DeepRacer car.
3. Evaluate performance on a race track.
4. Improve the model through additional training.

# 3. Key Concepts

- Reinforcement Learning (RL).
- Reward function.
- Autonomous driving.
- Simulation-based training.

# 4. Common Use Cases

- Learn Reinforcement Learning.
- Experiment with AI models.
- Participate in AWS DeepRacer competitions.

# 5. Benefits

- Hands-on RL experience.
- No prior robotics experience required.
- Virtual and physical racing environments.
- AWS-managed training infrastructure.
