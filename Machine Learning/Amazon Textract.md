# 1. Introduction

- Automatically extracts text, handwriting, and data from any scanned documents using AI and ML.
- Extract data from forms and tables.
- Read and process any type of document (PDFs, images, ...).
- **Use cases**
  - Financial Services (e.g., invoices, financial reports).
  - Healthcare (e.g., medical records, insurance claims).
  - Public Sector (e.g., tax forms, ID documents, passports).

# 2. Confidence Scores

- **Confidence scores** are values returned by **Amazon Textract** that indicate how confident the service is in the accuracy of the extracted text or detected information.

## 2.1. Purpose

- Assess the reliability of extracted data.
- Validate OCR results.
- Identify low-confidence extractions for review.
- Support quality assurance and human verification.

## 2.2. Common Use Cases

- Document processing.
- Invoice and receipt extraction.
- Form processing.
- Identity document analysis.

## 2.3. Benefits

- Measures extraction accuracy.
- Helps identify potential errors.
- Enables automated validation workflows.
- Improves confidence in document processing.
