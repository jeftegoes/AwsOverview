# 1. Introduction

- Fully managed **time-series forecasting** service.
- Uses **statistical methods** and **machine learning**.
- Predicts future values using historical data.
- No machine learning expertise required.

- Inventory planning.
- Workforce planning.
- Financial forecasting.
- Web traffic prediction.
- Server capacity planning.

# 3. Benefits

- Highly accurate forecasts.
- Fully managed by AWS.
