# 1. Introduction

- Natural and accurate **language translation**.
- Amazon Translate allows we to **localize content** - such as websites and applications - for **international users**, and to easily translate large volumes of text efficiently.

# 2. Custom Terminology

- **Custom Terminology** is an Amazon Translate feature that allows organizations to define **custom translations** for specific words, phrases, brand names, or industry-specific terms to ensure consistent and accurate translations.
- **Purpose**
  - Preserve brand names.
  - Translate industry-specific terminology correctly.
  - Ensure consistent translations across documents.
  - Improve translation accuracy.
