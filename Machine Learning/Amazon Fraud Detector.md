# 1. Introduction

- **Amazon Fraud Detector** is a **fully managed machine learning service** that helps detect **online fraud** in real time by using ML models, predefined rules, and historical event data.
- **Purpose**
  - Detect fraudulent activities.
  - Reduce financial losses.
  - Improve fraud detection accuracy.
  - Minimize manual fraud reviews.

# 2. Dataset Storage

- Amazon Fraud Detector allows datasets to be stored in **two ways**:
  1. **Internal Storage** (managed by Amazon Fraud Detector).
  2. **External Storage** (Amazon S3).
- The choice depends on the **model type**, dataset size, and data management requirements.

# 3. Feature Engineering

- **Feature Engineering** in Amazon Fraud Detector is the process of **automatically extracting and transforming relevant attributes** from raw event data to improve the accuracy of fraud detection models.
- **Purpose**
  - Automatically extract meaningful features.
  - Transform raw data into model-ready inputs.
  - Improve fraud detection accuracy.
  - Reduce manual data preparation.
