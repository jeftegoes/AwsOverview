# 1. Introduction

- Crowdsourcing marketplace to perform simple human tasks
- Distributed virtual workforce
- **Example**
  - We have a dataset of 10,000,000 images and we want to labels these images.
  - We distribute the task on Mechanical Turk and humans will tag those images.
  - We set the reward per image (for example $0.10 per image).
- **Use cases:** Image classification, data collection, business processing.
