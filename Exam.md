# 2. AWS Certified DevOps Engineer - Professional

## 2.1. Exam Time Management

- **Total time**: 180 minutes ( + 30 minutes if ESL is applied)
- **Total questions**: 75
- **Time per question**:
  - ~2.4 minutes per question (180 ÷ 75)
